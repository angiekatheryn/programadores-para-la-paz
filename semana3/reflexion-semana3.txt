Reflexión - Semana 3

¿Por qué es importante que los proyectos de software mantengan un registro de cambios usando herramientas como Git?

Es importante que los proyectos de software mantengan un registro de cambios con herramientas como Git porque permite llevar un control de las modificaciones realizadas, identificar quién hizo cada cambio y recuperar versiones anteriores en caso de errores. Esto ayuda a mantener el proyecto organizado y seguro.

Además, Git facilita el trabajo colaborativo entre varios desarrolladores, ya que todos pueden aportar cambios sin perder información. Gracias al historial de versiones, es más sencillo revisar, corregir y mejorar el software de manera eficiente.

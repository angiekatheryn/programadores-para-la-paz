Conceptos de Git - Semana 3

Repositorio:

Un repositorio es el espacio donde se almacenan los archivos de un proyecto y se registran los cambios realizados en ellos. Permite organizar el trabajo y mantener un historial de las diferentes versiones del proyecto para facilitar su seguimiento y colaboración.

Commit:

Un commit es un registro que guarda los cambios realizados en un momento específico del proyecto. Cada commit funciona como una fotografía del estado de los archivos, lo que permite identificar modificaciones y recuperar versiones anteriores si es necesario.

Push:

El comando push se utiliza para enviar los cambios guardados en el repositorio local hacia un repositorio remoto, como GitHub. Esto permite actualizar el proyecto en línea y compartir los avances con otros colaboradores.
